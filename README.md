# Gerenciamento de Pets e Cuidadores

Aplicativo feito com Ionic e Firebase para gerenciar cadastro de pets e cuidadores. Permite criar, listar, editar e excluir registros com uma interface simples e responsiva.

## Funcionalidades
- Cadastro e listagem de pets (nome, espécie, raça, idade, observações)
- Cadastro e listagem de cuidadores (nome, telefone, experiência, especialidades)
- Atualização e exclusão para pets e cuidadores
- Dados armazenados no Firebase Firestore

## Como usar

1. Clone o repositório:
git clone https://github.com/AlvesKZ/petshop.git

2. Instale as dependências e inicie o Ionic:
npm install
ionic init

3. Configure o Firebase (adicione suas credenciais no arquivo de ambiente).

4. Execute o app:
ionic serve
